# Лабораторная работа 4

**Работу выполнил:** Дубчак Александр, **группа:** 102.

**Постановка задачи:** Построить упорядоченное дерево для хранения и поиска элементов с
уникальными строковыми ключами и вещественными значениями. Ключ –
непустая последовательность латинских букв не более чем из 6 символов.
Ключи упорядочиваются лексикографически.
Виды деревьев:

1. AVL-дерево
2. В-дерево порядка 2 (n=2)
3. Красно-черное дерево
   Вид дерева для реализации - (N%3)+1, где N – номер в группе
   Для каждого вида дерева реализовать следующие 4 операции:
1. Добавление узла к дереву по ключу (с балансировкой при
   необходимости).
2. Удаление узла из дерева по ключу (с балансировкой при
   необходимости)
3. Печать дерева
4. Поиск значения элемента по заданному ключу.
   Предусмотреть возможность ввода команд из текстового файла в формате
   <номер операции> <ключ> <значение >
   Необходимость параметров в квадратных скобках определяется номером операции
5. В соответствии с номером 6 был выбран вариант: AVL-дерево

**Описание лабораторной работы:**

- `README.md(этот файл)` - файл, содержащий постановку задачи, краткое описание работы программы и сделанные выводы.

- `src` - папка, содержащая файлы, описывающие основной функционал программы.

    - `main.c` - файл, содержащий точку входа в программу, обработку ввода выражения и вывод результатов работы
      программы.

    - `add_node.c` - файл программы, реализующей добавление узла в дерево.

    - `balance.c` - файл программы, реализующей балансировку перекошенного дерева.

    - `bin_search.c` - файл программы, реализующей бинарный поиск узла в дереве по ключу.

    - `create_node.c` - файл программы, реализующей создание узла дерева, который содержит ключ-строку, значение-число,
      высоту в дереве, ссылку на левого и правого детей.

    - `delete_node` - файл программы, реализующей удаление узла дерева по ключу.

    - `get_height.c` - файл программы, реализующей вывод текущей высоты узла дерева.

    - `left_in_right.c` - файл программы, реализующей поиск самого левого элемента в правом поддереве узла,
      передаваемого в функцию node *left_in_right(node *root).

    - `print_tree.c` - файл программы, реализующей печать всего дерева.

    - `turns.c` - файл программы, реализующей повороты дерева при балансировке (LL, RR, RL, LR).

- `include` - папка, содержащая заголовочные файлы с объявлениями структур и функций программы.

- `run_tests` - папка, содержащая файлы с тестами для отдельных функций программы.

**Запуск программы:** для запуска программы нужно в терминале, в папке проекта, ввести gcc src/*.c -Iinclude -o nigga,
затем ввести
./nigga и выбрать действие в программе, а также необходимые аргументы в этой же строке через пробел при необходимости,
программа автоматически выполнит действие. Для выхода из программы нужно ввести **5**.

**Пример работы программы:**

    - Ввод:

    1 ye 5

    3

    - Вывод программы:

    Key: ye, Value: 5.000000, Height: 1
