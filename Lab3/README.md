# Лабораторная работа 3

**Работу выполнил:** Дубчак Александр, **группа:** 102.

**Постановка задачи:** составить программу на языке Си с использованием процедур и функций для разбора арифметического
выражения, представленного в инфиксной форме, построения его дерева разбора и приведения подобных слагаемых.  
Программа должна преобразовывать выражение в постфиксную форму (обратную польскую запись), строить по ней бинарное
дерево выражения и выполнять обход дерева для выделения слагаемых.  
В процессе обработки необходимо привести подобные слагаемые, то есть объединить элементы с одинаковой структурой и
вычислить итоговые коэффициенты.  
Для каждого выражения необходимо вывести промежуточные результаты и итоговое упрощённое выражение.  
В процессе отладки и тестирования рекомендуется использовать команды обработки текстовых файлов ОС UNIX и переадресацию
ввода-вывода. Тестовые данные необходимо заранее поместить в текстовые файлы.  
В качестве тестов можно использовать арифметические выражения с переменными, степенями и операциями сложения, вычитания
и умножения.

**Описание лабораторной работы:**

- `README.md(этот файл)` - файл, содержащий постановку задачи, краткое описание работы программы и сделанные выводы.

- `src` - папка, содержащая файлы, описывающие основной функционал программы.

    - `main.c` - файл, содержащий точку входа в программу, обработку ввода выражения и вывод результатов работы
      программы.

    - `dijkstra.c` - файл, реализующий преобразование инфиксного выражения в постфиксную форму (обратную польскую
      запись).

    - `build_free_tree.c` - файл, содержащий функции построения дерева выражения по постфиксной записи и освобождения
      памяти
      дерева.

    - `create_node.c` - файл, содержащий функцию создания узлов дерева выражения.

    - `add.c` - файл, содержащий функции обработки слагаемых (сбор, разбиение и объединение подобных членов).

- `include` - папка, содержащая заголовочные файлы с объявлениями структур и функций программы.

- `run_tests` - папка, содержащая файлы с тестами для отдельных функций программы.

**Запуск программы:** для запуска программы нужно в терминале, в папке проекта, ввести ./build/Lab3, затем ввести
арифметическое выражение в инфиксной форме. Программа автоматически выполнит преобразование выражения, построит дерево и
выведет результат упрощения. Для выхода из программы нужно ввести **^C**.

**Пример работы программы:**

    - Ввод:

    3-1*x^2+2*x^2

    - Вывод программы:

    Postfix: 3 1 x 2 ^ * - 2 x 2 ^ * +

    Result: 3+(x^2)

**Выводы:** В ходе выполнения лабораторной работы была разработана программа на языке C для разбора арифметических
выражений и приведения подобных слагаемых. В работе использовался алгоритм Дейкстры для преобразования выражения в
постфиксную форму и построение бинарного дерева выражения для дальнейшей обработки.

В качестве структуры данных было выбрано бинарное дерево, где каждый узел представляет либо оператор, либо операнд. Это
позволило удобно выполнять обход выражения и анализировать его структуру.

В программе реализованы следующие функции:
• преобразование инфиксной записи в постфиксную;

	•	построение дерева выражения;

	•	обход дерева для выделения слагаемых;

	•	разбиение выражений на коэффициенты и базовые части;
    
	•	объединение подобных слагаемых.

Алгоритм обработки выражения позволяет выделять одинаковые по структуре части выражения и складывать их коэффициенты. В
результате тестирования программы было проверено корректное выполнение упрощения выражений. После обработки выражение
приводится к более компактному виду за счёт объединения подобных слагаемых.

Таким образом, поставленная задача была выполнена: реализован алгоритм разбора выражения, построения дерева и приведения
