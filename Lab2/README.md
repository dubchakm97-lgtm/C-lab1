# Лабораторная работа 2

**Работу выполнил:** Дубчак Александр, **группа:** 102.

**Постановка задачи:** cоставить программу на языке Си с использованием процедур и функций для сортировки таблицы заданным методом и двоичного поиска по ключу в таблице. 
Программа должна вводить значения элементов неупорядоченной таблицы и проверять работу 
процедуры сортировки в трех случаях: (1) элементы таблицы с самого начала упорядочены; (2) элементы 
таблицы расставлены в обратном порядке; (3) элементы таблицы не упорядочены. В последнем случае 
можно использовать встроенные процедуры генерации псевдослучайных чисел. 
Для каждого вызова процедуры сортировки необходимо печатать исходное состояние таблицы и 
результаты сортировки. После выполнения сортировки программа должна вводить ключи и для каждого из 
них выполнять поиск в упорядоченной таблице с помощью процедуры двоичного поиска и печатать 
найденные элементы, если они присутствуют в таблице. 
В процессе отладки и тестирования рекомендуется использовать команды обработки текстовых 
файлов ОС UNIX и переадресацию ввода-вывода. Тестовые данные необходимо заранее поместить в 
текстовые файлы. 
В качестве тестов можно взять фрагменты текста (стихи или проза).

**Описание лабораторной работы:**
- `README.md(этот файл)` - файл, содержащий постановку задачи, краткое описание работы программы и сделанные выводы.
- `main.c` - программа, составленная на языке программирования С, содержащая следующие функции:
    • `typedef struct Chart` - структура одной ячейки, в которой вместе хранятся ключ (строка) и значение (число).
    
    • `void sort` - функция сортировки таблицы, состоящей из множества ячеек count

    • `int main` - точка входа в программу

- `tests` - папка, содержащая 5 файлов с тестовыми данными, а также 5 файлов предполагаемого вывода заданных тестов

- `Results` - папка, содержащая 5 файлов, в которых хранится результат вывода программы на n-ом тесте


**Запуск программы:** для запуска программы нужно в терминале программы, в папке C-lab1, ввести ./build/Lab2, а затем ввести стартовое значение - количество ключей в таблице. Затем последовательно можно вводить ключи (строки) и значения (числа). Для выхода из программы нужно ввести **^C**.

**Выводы:** В ходе выполнения лабораторной работы была разработана программа на языке C для сортировки таблицы и поиска элементов по ключу. В работе использовался метод двоичной вставки (Binary Insertion Sort) для упорядочивания таблицы и двоичный поиск для нахождения элементов по ключу.

В качестве структуры таблицы был выбран вариант 4 — строковый ключ с совместным хранением ключа и данных в одной структуре. Каждый элемент таблицы содержит строковый ключ и связанное с ним числовое значение.

В программе реализованы следующие функции:
	•	ввод элементов таблицы;

	•	вывод таблицы до и после сортировки;

	•	сортировка элементов методом двоичной вставки;
    
	•	поиск элементов в отсортированной таблице с помощью двоичного поиска.

Алгоритм двоичной вставки позволяет уменьшить количество сравнений при поиске позиции для вставки элемента по сравнению с обычной сортировкой вставками. Для определения позиции используется двоичный поиск, после чего элементы массива сдвигаются вправо и новый элемент вставляется в найденную позицию. В результате тестирования программы было проверено корректное выполнение сортировки. После выполнения алгоритма таблица была упорядочена по строковому ключу в лексикографическом порядке. 

Таким образом, поставленная задача была выполнена: реализован алгоритм сортировки таблицы методом двоичной вставки и алгоритм двоичного поиска по ключу. Программа корректно обрабатывает входные данные, выполняет сортировку и обеспечивает эффективный поиск элементов в отсортированной таблице.
