# Лабораторная работа 1

**Работу выполнил:** Дубчак Александр, **группа:** 102.

**Постановка задачи:** Составить программу на языке Си для построения и обработки дерева общего вида или упорядоченного 
двоичного дерева, содержащего узлы типа iпt. Основные функции работы с деревьями реализовать в виде отдельных 
функций. После того, как дерево создано, его обработка должна производиться в режиме текстового меню со 
следующими действиями: 
• добавление нового узла (для двоичного дерева положение нового узла определяется в соответствии с требованием 
сохранения порядка; для дерева общего вида должен задаваться родитель добавляемого узла. Добавляемый узел 
становится самым младшим потомком); 
• текстовая визуализация дерева (значение каждого узла выводится в отдельной строке, с отступом, 
пропорциональным глубине узла, в порядке старшинства узлов); 
• удаление узла (двоичное дерево перестраивается в соответствии с требованием сохранения целостности и порядка; 
для дерева общего вида удаляется все поддерево, исходящее из удаляемого узла. Должно быть предусмотрено 
корректное освобождение памяти);
• определить значение нетерминальной вершины двоичного дерева с максимальной глубиной;

**Описание лабораторной работы:**
- `README.md(этот файл)` - файл, содержащий постановку задачи, краткое описание работы программы и сделанные выводы.
- `main copy.c` - программа, составленная на языке программирования С, содержащая следующие функции:
    • `typedef struct Node` - структура узла дерева
    
    • `node *add` - функция добавления узла с задаваемым значением value

    • `void delete` - удаление узла с подаваемым значением

    • `void traversal_lkp` - обход и печать узлов дерева по принципу ЛКП

    • `void traversal_klp` - обход и печать узлов дерева по принципу КЛП

    • `void traversal_lpk` - обход и печать узлов дерева по принципу ЛПК

    • `void mdv` - поиск и вывод нетерминального узла, имеющего максимальную глубину

    • `void print` - поиск и печать узла с задаваемым значением и его детей

    • `int main` - точка входа в программу
    
- `tests` - папка содержащая 5 файлов с тестовыми данными, а также 5 файлов предполагаемого вывода заданных тестов


**Запуск программы:** для запуска программы нужно в терминале программы, в папке C-lab1, ввести ./cmake-build-debug/untitled, а затем ввести стартовое значение - значение кореня дерева. Затем последовательно можно вводить команды add, delete, print, traversal(ЛКП, КЛП, ЛПК), mdv, функционал которых описан выше. Для выхода из программы нужно ввести **exit**.

**Выводы:** в ходе выполнения лабораторной работы была разработана программа на языке Си для построения и обработки упорядоченного двоичного дерева.

В программе реализованы основные операции работы с деревом:

• добавление нового узла

• удаление узла

• обход дерева (ЛКП, КЛП, ЛПК)

• вывод информации об узле

• вычисление функции от дерева

Добавление узлов осуществляется с сохранением свойства упорядоченности двоичного дерева. Удаление узлов производится с корректным освобождением памяти.

В соответствии с вариантом №6 была реализована функция определения значения нетерминальной вершины двоичного дерева, имеющей максимальную глубину. Для решения задачи использован рекурсивный обход дерева, в процессе которого определяется глубина каждой нетерминальной вершины и выбирается вершина с максимальной глубиной.

Разработанная программа работает в режиме CLI (текстового меню) и позволяет пользователю выполнять операции над деревом в интерактивном режиме.

В результате выполнения лабораторной работы были закреплены навыки:

• работы с динамическими структурами данных

• реализации бинарных деревьев

• использования указателей и динамического выделения памяти

• применения рекурсии для обработки деревьев
